    public class SparqlPreprocessor : SparqlTokeniser, ITokeniser
    Remarks

    This class is primarily intended to support a limited range of query preprocessing tasks such as setting the values of query parameters (@-variables) as quickly as possible. It does not aim to support the full SPARQL standard syntax.
