    Class SparqlQuery

    A simple SPAQL Query class. This class aims to ease formulating SPARQL query strings by automatically setting PREFIX declarations from a given namespace manager.

    SparqlQuery(String, Boolean)

    Creates a new SPARQL query. If enabled, the PREFIXES used in any of the query's graph patterns will be declared in the query header if they are found in the application config. Additionally, the query may be compacted in order to reduce processing overhead when being used repeatedly in loops.
